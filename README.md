# Modul 13 - Git dan GitHub

Proyek ini digunakan untuk latihan dasar kontrol versi menggunakan Git dan GitHub.

## Deskripsi

Proyek web sederhana yang dibuat sebagai bagian dari praktikum Pemrograman Web 1, Modul 13, untuk mempelajari penggunaan Git dan GitHub dalam pengembangan web secara kolaboratif.

## Fitur

- Struktur HTML sederhana
- Styling CSS
- Interaksi JavaScript sederhana

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript
- Git dan GitHub

## Cara Menjalankan Proyek

1. Clone atau unduh repository ini.
2. Buka file `index.html` menggunakan browser.
3. Klik tombol "Ubah Deskripsi" untuk melihat interaksi JavaScript.
